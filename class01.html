2開場:CSR / ESG 與利潤極大化

討論主軸: 企業做 CSR / ESG 到底跟利潤衝突嗎?

  • 兩派學術觀點(至今無定論):
    • 正相關派:投入 CSR → 提升企業獲利
    • 非線性派:先助益 → 動化 → U 型 / 倒 U 型都有
  • 老師立場(古典經濟學派): 不管加什麼面向,最終仍以 dollar sign 來體現
  • CSR / ESG 對某些企業是「必要性調整」 — 例如台灣半導體要進國際供應鏈,不做 ESG 客戶就不下單 → 它就是利潤模式的一部分
  • 對某些企業不是必要,那時候花資源做 CSR 反而侵蝕獲利

4外部環境分析工具

4.1 Porter 五力分析(Five Forces)

分析層級: 產業層級(不是市場、也不是公司)

五力:

  1. 現有競爭者(rivalry)
  2. 潛在進入者(new entrants,還沒進來但你聞到消息)
  3. 供應商議價力
  4. 買方議價力
  5. 替代品威脅
🔑 關鍵陷阱:Scope 的定義決定結果

老師與隆志學長(能源電池產業)現場對話:

  • 隆志的公司做:AI 資料中心電池 / 電梯電池 / 車輛電池 3 種
  • 老師問:「你怎麼定位你們公司的產業?」
  • 結論:同一家公司若面對多個市場,可能需要做 3 份不同的 Porter 五力分析

原則: Scope 越大 → 涵蓋的競爭者越多,分析越攏統;Scope 越明確 → 只看跟你一模一樣的公司,更精準。每個分析者定位的產業 scope 都不一樣,這是主觀決策。

🚗 Tesla 案例(現場真實)

同一款 Model,在不同國家獲利模式完全不同:

地區獲利來源
歐盟賣車 + 競爭對手要花錢跟 Tesla 買碳權(Tesla 一台車能有 100% 碳權收益)
台灣只能從消費者身上一分一分辛苦賺 → 政府不承認 Tesla 的碳權

教訓: 同樣產業,政治法規 + 民情差異 導致獲利模式差異巨大。做五力分析時 country context 是隱藏變數。

4.2 Porter 鑽石理論(Diamond Model)

分析層級: 國家層級(不是產業)
書: Michael Porter, The Competitive Advantage of Nations(1990 年初)

四大關鍵要素:

要素內涵
1. 生產要素天然資源、勞力、土地、資本(國家天生條件差異)
2. 需求條件國內市場對這個產品的需求性 · 例:歐盟碳權才能賣,不做綠色就進不去
3. 相關與支援產業產業聚落 · 台灣半導體聚落 vs 歐美散開
4. 企業策略、結構、競爭國內競爭態勢

5VRIO / RBV 資源基礎理論 ★

作者: Jay Barney(1991)
位階: RBV = Resource-Based View,英文用 View 而不是 Theory(因為驗證困難,至今仍是「觀點」)

5.1 資源要具備的四個特徵(VRIO)

V - Valuable         有價值
R - Rare             稀少
I - Imperfectly      不可(完美)模仿
    Imitable
O - Organization     組織能運用(原本是 N: Non-substitutability,不可取代)
  • 都具備 → 走到「Sustained Competitive Advantage(持續性競爭優勢)」機率高
  • 只有前幾項 → 只是 Survival(留在市場,不會被踢出)

5.2 每個特徵怎麼判斷

特徵判斷方式例子
V (Valuable)能否提升效率、降低成本、或激盪出跟外部環境的獨特互動台積電有專利、產業聚落
R (Rare)別人有沒有?我有他們沒有?台積電的先進製程
I (Imitable)不可被模仿 · 3 個原因(見下)
O (Organization)有能力把資源轉成競爭力組織文化能否運用

5.3 為什麼資源不容易被模仿?三個原因(重點!)

  1. Unique Historical Conditions(獨特的歷史條件)
    • 每家公司成長歷史都不一樣,對手學不來
  2. Causal Ambiguity(因果模糊性)
    • 光台積電成功要素就有 4–5 條,每一條都要學嗎?每一條都不學嗎?
    • 當事人自己也搞不清楚是哪個資源導致成功 → 局外人更學不會
  3. Social Complexity(社會複雜性)
    • 組織內部的人際信任、玩力者之間、員工之間的關係
    • 200 人的組織,人際關係複雜性外人幾乎複製不了

5.4 現場討論:哪個特徵最重要?

  • 有人說 V 最重要(沒 V,其他三個沒意義)
  • 有人說 I 最重要(不可模仿才能持久)
  • 有人說 四個缺一不可
老師結論 「學界公說公有理,學術上真的沒有定論。」

5.5 新創公司適用嗎?(隆志學長現場提問)

  • 隆志問: 新創沒有這些資源啊,我要怎麼用 VRIO 評估?
  • 佩紅(創投)答: 我們創投沒那麼學術,但採量都是四大頂
  • 老師回: 新創的資源不是憑空來的,一定是前面一步一步累積出來 → VRIO 也適用,只是你看到的階段是後端
  • 創投輔導的角色: 幫新創補足資源(intel value、發行能力、退場管道)

5.6 老師血淚提醒:RBV 的兩個限制

  1. 是靜態的(照張相 · 只有那個 timing)→ 後來才有 Dynamic Capabilities(動態能力)接續
  2. 沒教你怎麼「創造」新資源(只教你怎麼審視既有的)

5.7 學術上怎麼驗證 RBV?

極度困難,老師直言:

  • 資源難以量化(社會複雜性、因果模糊)
  • 資料難以收集(要回顧 20 年前的環境)
  • 案例研究可以,但很難形成量化 empirical evidence
  • 統計要求大量本:30 個樣本不夠,產業研究常常要幾千比
為什麼 RBV 至今仍是「View」不是「Theory」 因為在管理學無法用量化驗證。這是本質限制,不是學者不努力。

6Business Level Strategy(事業層級策略)

6.1 核心經濟模型:三條線

Value (V) — 消費者知覺價值
──────── 消費者剩餘 = V − P ────────
Price (P) — 消費者付的價格
──────── 企業利潤 = P − C ────────
Cost (C) — 企業成本

利潤極大化就兩條路徑:

路徑做什麼稱為
拉高 V差異化你的產品/服務,讓消費者願意付更多Differentiation(差異化)
壓低 C降低成本Cost Leadership(成本領導)

6.2 進階討論:能兼顧兩個嗎?

老師拋出:「能不能兩個都拿?」

現場答案:

  • 主流文獻建議:「魚與熊掌不可兼得,要擇一 mainly」
  • 但很多時候可以兼顧(降低成本同時做廣告拉高價值)
  • 老師的實用建議: 你能做到就做,但主副仍要決定

6.3 兩個進階操作(超越簡單模型)

  • 壓低 C 但 V 只小降 → 面積可能還變大(蘋果做過)
  • 投入新技術 → 整個 MC 曲線下移 → V/C 都變(這是老師講的「打通任督二脈」的方法)

6.4 隆志學長的移動屋案例

  • 他的產品: 移動式房屋(鋼構、雷軌),可拆組、可移動
  • 成本: 一平約 10 萬,30 平 = 300 萬(遠低於一般 3 房 1600 萬)
  • 市場定位討論:
    • 不是取代主流建物(客戶不會 10 年一直住)
    • 是取代營造廠工地宿舍、民宿、辦公室、行動招待所、退休移動生活
  • 與破壞式創新的關係: 見下節

7Christensen 破壞式創新 ★

作者: Clayton Christensen (Harvard Business School)

老師分享的內幕 Christensen 是位非常好的老好先生,是農人教徒(阿米許)。有台灣同學曾去哈佛旁聽他的課,他的兒子曾來台灣傳教(騎腳踏車那個教派)。

7.1 破壞式創新的定義(現場學生要背下來!)

兩個必要條件:

  1. 切入 low-end / new market
    • 一開始被主流廠商瞧不起、沒注意到
    • 品質夠用就好(不追求 high-end)
    • 用比較低價取得
  2. 吸引原本沒在使用的人
    • 從無到有(true new market)
    • 或原本用不起的人現在用得起

完成條件: 最後取代主流市場的客戶才算完成整個過程。

7.2 vs 持續性創新(Sustaining Innovation)

破壞式持續性
低價低品質切入用既有資源做增量創新
新市場 / 被忽略的市場現有市場
一開始沒有威脅感已知賽道

7.3 現場舉的例子

例子是否算破壞式?老師判斷
Uber 叫車✅平台化,不用電話,取代計程車行
Nokia → 智慧手機✅破壞掉 Nokia 主流
相機 → 手機拍照✅底片沒了
東林東(移工便當)✅忽略市場,只做移工便當費 + 匯率
豪泰(節能燈具)✅業主 0 成本換燈具,拆帳電費
移動屋(隆志)現階段持續性還沒取代主流,但有機會成為破壞式

7.4 現場學生問:「東林東為什麼算?」

  • 人力仲介都靠扣移工錢賺
  • 東林東:完全不扣移工的錢
  • 賺兩件事:
    • 便當費(自家另一間公司負責出便當)
    • 匯率(移工匯款回國)
老師點評 「這個名字很棒,還蠻貼切後來是創新的概念,就完全不是跟對手打的東西。」

8Corporate Level Strategy(公司層級策略)

8.1 多角化 Diversification

兩種:

  • 產業多角化 — 離開原本 Value Chain 階段,往其他階段走
  • 區位多角化 — 進入不同國家 / 區域市場

學術發現: 多角化程度 vs 績效關係 → 正、負、U 型、倒 U 型都有 → 沒定論 → 因產業而異

8.2 Vertical Integration(垂直整合)/ 一條龍

  • 上游 / 下游整合
  • 100% 垂直整合 = 一條龍
  • 鴻海是經典(價值鏈 5 個階段自己都拿)

10交易成本理論 ★★★

作者: Ronald Coase(1937 原始論文)、Oliver Williamson(現代化 · 諾貝爾獎)
諾貝爾獎: 至少 3 位靠這理論拿到經濟學諾獎
位階: 老師說「這個理論的重要性,更甚至超過資源基礎理論」

10.1 經典例子:蜜蜂與果園

  • 養蜂人跟果園主吵架:誰該付誰錢?
  • 養蜂:「我蜜蜂飛去你果園受粉,你要付我!」
  • 果園:「我花朵給你蜜蜂採蜜,你要付我!」
  • 兩邊都有道理 → 討價還價 → 攀價需要付出極高的交易成本
交易成本 定義 為完成交易所需付出的所有無形成本(協商、討價、契約、追蹤、風險)。

10.2 人性兩個基本假設(不接受 → 整個理論不成立)

  1. 機會主義(Opportunism) — 人性本惡,有機可乘就會投機
  2. 有限理性(Bounded Rationality) — 人不是算命師,預測不了所有未來

→ 導致契約必然不完整 → 有需要就要重新議約 → 又是交易成本

10.3 三個決定「交易成本高低」的因素 ★★★

因素定義高的時候 → 怎麼辦
1. 資產專屬性 (Asset Specificity)我的投入只對特定對象有用(電動車電池對 Tesla 特別)被套牢 → 弱勢 → 提高交易成本 → 考慮 Merge/Buy
2. 交易頻率 (Frequency)越頻繁需要訂約交易成本累積 → 考慮自製
3. 環境不確定性 (Uncertainty)產業/國家環境變動大對方投機機率高 → 議約更小心 → 考慮 Merge

10.4 應用:Make or Buy?

核心邏輯:

  • 交易成本很高 → 自己做(Make / Vertical Integrate / Merge)
  • 交易成本很低(供應商多、標準化、專屬性低)→ 外購(Buy / Market)

10.5 為什麼會有企業(FIRM)?

  • 原始人時代 → 各自跑單幫,每筆交易都討價還價 → 交易成本累積驚人
  • 有 FIRM(公司)這樣的組織 → 集合一群人,內部不用一直訂約 → 大幅降低交易成本
所以企業存在的本質是「降低交易成本」。

10.6 更多應用場景

情境用交易成本理論怎麼想
進入海外市場自己設子公司 vs 品牌授權當地
要不要併購交易成本極高、資產專屬高、無法自行發展 → 併購
買 vs 自製供應商多、標準化 → 買;反之 → 自製
併購金額100 億買 → 拿到的溢價能力可能到 1000 億(時間差、無法自行發展、規模效益)

10.7 學術限制

  • 交易成本難以量化(它是無形的)
  • 只能觀察「達成合約的難易度」反推
  • 老師舉例:「他(某公司)也在挖了幾百億」— 那不是交易成本,那是溢價的購買金額;交易成本是額外付出達成合約的努力

10.8 併購三大關鍵決策因素(佩紅學姊補充)

  1. 時間的考量 — 快速切入 window
  2. 無法自行發展(No make it in-house)— 對方有你複製不了的特點
  3. 規模效益 — 併購後產生 1+1 > 2

12老師反覆強調的心法(易踩雷)

  1. 管理學沒有標準答案,但必須有一套邏輯論述可以說服別人。同樣議題可以「公說公有理、婆說婆有理」,關鍵是論述的嚴謹度與說服力。
  2. 每個理論背後都有一個 research question — 找論文題目就從「這理論想解決什麼問題」開始想。
    • RBV 想回答:為什麼有些企業有競爭優勢?
    • TCE 想回答:為什麼有企業(FIRM)這樣的組織形態?
  3. 策略 ≠ Mission/Vision 的話術 — 策略是利潤極大化的 gameplay,Mission/Vision 只是傳達方式
  4. Porter 五力的 Scope 是主觀決策 — 同一家公司可能要做多份分析
  5. VRIO 不可模仿的三個來源:歷史條件、因果模糊、社會複雜 — 這是 RBV 最深的部分
  6. 破壞式創新有兩個條件缺一不可:低端切入 + 吸引新用戶。單有低價 ≠ 破壞式
  7. 交易成本的三個因素:資產專屬性、交易頻率、環境不確定性 — 三個都高 → 該併購
  8. 併購金額 ≠ 交易成本(佩紅提問的點)— 交易成本是額外付出的溝通/協商/風險成本
  9. 「你們是 DBA、有產業經驗,你們的實務經驗可能比我們學者的理論更真實」 — 學者是紙上談兵
  10. 管理學是社會科學,不是 Hard Science — 很多論述沒辦法量化驗證,這是本質限制

13老師分享的學界內幕

  • Christensen(破壞式創新)人在世,是農人教徒,兒子曾來台灣傳教
  • 資源基礎理論的 Jay Barney 是代表性學者,但不是唯一,是一大群學者陸陸續續 20–30 年建構的
  • 交易成本理論 → 至少 3 位諾貝爾經濟學獎(Coase、Williamson、還有一位)
  • 學術路徑上,同一議題會有多派學者互補、互相演化(RBV → Dynamic Capabilities)
  • View vs Theory 的英文差別:能被量化驗證的才叫 Theory,不能的只能叫 View / Perspective

1530 秒口頭複習

今天走完策略管理的整套理論地圖:

外部先看 — Porter 五力(產業層級)+ 鑽石(國家層級);
內部再看 — RBV / VRIO 四要素,不可模仿源自「歷史 · 因果模糊 · 社會複雜」;
策略選擇 — 事業層級走差異化 vs 成本領導;
創新路徑 — Christensen 破壞式(低端 + 新用戶);
公司層級 — 多角化、垂直整合、一條龍;
自製 vs 外購 — 交易成本三因素(資產專屬、頻率、不確定性)決定。

管理學沒有標準答案,只有「論述有沒有說服力」。 每個理論背後都有一個 research question,那就是找論文題目的起點。你們 DBA 學生的實務經驗 + 理論框架 = 貢獻的原點。
